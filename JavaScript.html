JavaScript

JavaScript es un lenguaje de programación orientado a objetos que permite implementar funciones complejas en páginas web para que sean dinámicas. Es compatible con todos los navegadores modernos y se ejecuta en el navegador del usuario. Es la tercera capa de las tecnologías web estándar, junto a HTML y CSS.

Usos de JavaScript

JavaScript es posiblemente el lenguaje de programación más popular de la web, siendo actualmente usado para:

  • Interactividad y dinamismo en páginas web
  • Back-end de aplicaciones y sitios web
  • Aplicaciones web
  • Juegos de navegador

Cómo utilizar JavaScript en una página web

Para añadir JavaScript a una página web simplemente hay que usar la etiqueta script, de forma muy similar a como se vincula CSS con un documento HTML. Hay dos formas para lograr esto:

  1. Enlazando la etiqueta script a un archivo externo.
    2. <script src="tuArchivo.js"></script>
  2. Inyectando el código JavaScript directamente en el HTML.
    3. <script> Código JavaScript </script>

Se recomienda trabajar con la primera forma, ya que trabajar el código en un archivo separado permite modificarlo sea más fácil.

Manejo de variables en JavaScript

Declaración de variables

Hay cuatro formas de declarar variables en JavaScript:

  1. Automáticamente. No se recomienda por no ser buena práctica.
  2. Var. Usada en versiones antiguas de JavaScript.
  3. Let. La más recomendada actualmente.
  4. Const. Usada para valores constantes que no van a cambiar.
x = 5;
var y = 6;
let z = 7;
const pi = 3.14;

Uso de variables

Se puede mostrar el valor de una variable usando getElementById() para asignarle a un elemento específico el valor de dicha variable.

<p id="demo"> </p>
<script>
 let nombreCarro = "Volvo";
 document.getElementById("demo").innerHTML = nombreCarro;
</script>

También se pueden usar diferentes operadores aritméticos (+, -, *, /, %) para modificar los valores de las variables.

z = (x + y) / pi;

Ejemplo de JavaScript: IMC

El índice de masa corporal es el peso de una persona en kilogramos dividido por el cuadrado de la estatura en metros. El IMC es un método de evaluación fácil y económico para la categoría de peso: bajo peso, peso saludable, sobrepeso, y obesidad.

El IMC no mide la grasa corporal directamente, pero el IMC se correlaciona moderadamente con medidas más directas de la grasa corporal. Además, el IMC parece estar tan fuertemente correlacionado con diversos resultados metabólicos y de enfermedades como lo están estas medidas más directas de la grasa corporal.

La correlación entre el IMC y la grasa corporal es bastante fuerte, pero incluso si dos personas tienen el mismo IMC, su nivel de grasa corporal puede ser diferente. Si bien es muy probable que una persona con un IMC muy alto tenga una grasa corporal alta, un IMC relativamente alto puede ser el resultado de una grasa corporal alta o una masa corporal magra alta (músculos y huesos).
